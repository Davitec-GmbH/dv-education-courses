# Education Courses (dv_education_courses)

Generic course catalog for the education sector. Manages courses, events, locations, and contact persons.

## Features

- Course management with type categorization, contact persons, and images
- Event scheduling with single event dates, locations, and pricing
- Course list with search and type filter
- Course detail with upcoming events and similar course suggestions
- Event list and detail views
- TYPO3 v12 LTS and v13 LTS support

## Requirements

- PHP 8.2+
- TYPO3 v12.4 LTS or v13.4 LTS

## Installation

```bash
composer require davitec/dv-education-courses
```
